# Credex AI Audit

A modern AI Spend Audit dashboard built for the Credex Web Development Internship Assignment.

---

# Overview

Credex AI Audit is a frontend SaaS-style web application that helps startups and teams analyze their AI tool spending and discover optimization opportunities.

Users can:
- Add AI tools and subscriptions
- Analyze monthly AI spending
- Generate optimization insights
- View savings analytics and charts
- Explore AI audit reports
- Access responsive dashboards across devices

---

# Features

## Authentication
- Login & Signup UI
- LocalStorage-based authentication
- Dynamic user profile/avatar
- Logout functionality

## Dashboard
- AI spend overview cards
- Current spend tracking
- Optimized spend estimation
- Monthly & annual savings calculations

## AI Spend Configuration
- Add/remove AI tools
- Tool plans and pricing inputs
- Seat management
- Use-case categorization

## AI Audit Engine
- Rule-based optimization analysis
- Enterprise plan recommendations
- API usage optimization insights
- Cost reduction suggestions

## Analytics
- Doughnut chart for spend distribution
- Bar chart for cost comparison
- Analytics overview page

## Reports
- Audit report section
- Optimization summaries
- Smart recommendations

## Responsive Design
- Mobile responsive sidebar
- Hamburger menu navigation
- Modern SaaS UI/UX
